# Introduction

Welcome to **Hey Bible** - a free and easy to use Bible verse lookup tool with support for multiple translations.

![Hey Bible Banner](img/banner.png)

## Getting Started

With Hey Bible you can search for verses, save your favorites, add notes, and create beautiful, unique images of your favorite Bible verses to share with friends.

## For Developers

### API

Hey Bible provides a REST API for programmatic access to your saved verses, notes, and images. Generate an API key from your account settings to get started.

- [API Reference](/api) - Full OpenAPI documentation
- [Generating API Keys](/developers/generating-api-keys) - How to create and manage API keys
- [Node.js Client](/developers/nodejs-client) - Official Node.js SDK

### MCP Server

Connect Hey Bible to AI assistants like Claude Desktop using the Model Context Protocol (MCP). Access your saved verses and notes directly from your AI conversations.

- [MCP Server Setup](/developers/mcp-server) - Installation and configuration guide
